## 프로젝트 진행 방법
1. 해당 프로젝트를 fork하여 자신의 github에 추가한 뒤 그곳에서 프로젝트를 진행해주세요.
2. PBL을 진행할 수 있게 디렉토리를 구성해주세요.
3. 매 주차 작업이 끝난 뒤에는 pull request를 통해 본 레포에 업데이트합니다.

  3-1. PR을 진행할 때 main repo가 아닌 {name} 브랜치에 PR을 요청합ㄴ디ㅏ. ex) 강현우 브랜치에 PR 요청
